# 16 优化算法

优化算法决定神经网络如何更新参数。从基础的 SGD 到自适应的 Adam，选择合适的优化器对训练效果至关重要。

## 优化问题概述

### 为什么需要优化

神经网络训练本质上是一个**优化问题**：找到一组参数，使得损失函数最小。

$$\min_{\mathbf{\theta}} J(\mathbf{\theta})$$

### 优化的挑战

| 挑战 | 描述 | 影响 |
|------|------|------|
| **高维空间** | 参数维度可达数百万 | 搜索空间巨大 |
| **非凸性** | 损失函数有多个局部最优 | 可能陷入局部最优 |
| **鞍点** | 梯度为零但不是极值点 | 训练停滞 |
| **梯度消失/爆炸** | 梯度太小或太大 | 深层网络训练困难 |
| **病态条件** | 不同参数方向曲率差异大 | 收敛慢 |

### 优化的分类

| 类别 | 方法 | 特点 |
|------|------|------|
| **一阶方法** | SGD, Adam | 使用梯度，计算快 |
| **二阶方法** | 牛顿法, L-BFGS | 使用Hessian，收敛快但计算量大 |
| **自适应方法** | Adam, RMSprop | 自动调整学习率 |

## SGD 回顾

### 基本形式

$$\theta_{t+1} = \theta_t - \alpha \nabla J(\theta_t)$$

```python
import numpy as np

class SGD:
    def __init__(self, params, lr=0.01):
        self.params = params
        self.lr = lr
    
    def step(self, grads):
        for param, grad in zip(self.params, grads):
            param -= self.lr * grad
```

### 问题

1. **梯度方向震荡**：在峡谷地形中来回震荡
2. **学习率固定**：难以适应不同参数的变化速度

## Momentum

### 核心思想

引入"动量"，累积之前的梯度方向：

$$v_t = \beta v_{t-1} + (1-\beta) \nabla J(\theta_t)$$

$$\theta_{t+1} = \theta_t - \alpha v_t$$

```python
class Momentum:
    def __init__(self, params, lr=0.01, momentum=0.9):
        self.params = params
        self.lr = lr
        self.momentum = momentum
        self.velocities = [np.zeros_like(p) for p in params]
    
    def step(self, grads):
        for i, (param, grad) in enumerate(zip(self.params, grads)):
            self.velocities[i] = self.momentum * self.velocities[i] + grad
            param -= self.lr * self.velocities[i]
```

### 效果

- 加速收敛（下坡时累积速度）
- 抑制震荡（穿越峡谷时保持方向）

## Nesterov Accelerated Gradient (NAG)

### 核心思想

先按动量方向"看一步"，再计算梯度：

$$v_t = \beta v_{t-1} + \alpha \nabla J(\theta_t - \beta v_{t-1})$$

$$\theta_{t+1} = \theta_t - v_t$$

```python
class NAG:
    def __init__(self, params, lr=0.01, momentum=0.9):
        self.params = params
        self.lr = lr
        self.momentum = momentum
        self.velocities = [np.zeros_like(p) for p in params]
    
    def step(self, grads):
        for i, (param, grad) in enumerate(zip(self.params, grads)):
            # 先按动量方向更新
            param -= self.momentum * self.velocities[i]
            
            # 计算梯度
            self.velocities[i] = self.momentum * self.velocities[i] + grad
            param -= self.lr * self.velocities[i]
```

## AdaGrad

### 核心思想

自适应学习率：对频繁出现的特征使用小学习率，对稀疏特征使用大学习率：

$$G_t = G_{t-1} + (\nabla J)^2$$

$$\theta_{t+1} = \theta_t - \frac{\alpha}{\sqrt{G_t} + \epsilon} \nabla J$$

```python
class AdaGrad:
    def __init__(self, params, lr=0.01, eps=1e-8):
        self.params = params
        self.lr = lr
        self.eps = eps
        self.G = [np.zeros_like(p) for p in params]
    
    def step(self, grads):
        for i, (param, grad) in enumerate(zip(self.params, grads)):
            self.G[i] += grad ** 2
            param -= self.lr * grad / (np.sqrt(self.G[i]) + self.eps)
```

### 问题

学习率单调递减，可能过早停止。

## RMSprop

### 核心思想

使用移动平均替代累积平方：

$$G_t = \beta G_{t-1} + (1-\beta)(\nabla J)^2$$

$$\theta_{t+1} = \theta_t - \frac{\alpha}{\sqrt{G_t} + \epsilon} \nabla J$$

```python
class RMSprop:
    def __init__(self, params, lr=0.01, beta=0.9, eps=1e-8):
        self.params = params
        self.lr = lr
        self.beta = beta
        self.eps = eps
        self.G = [np.zeros_like(p) for p in params]
    
    def step(self, grads):
        for i, (param, grad) in enumerate(zip(self.params, grads)):
            self.G[i] = self.beta * self.G[i] + (1 - self.beta) * grad ** 2
            param -= self.lr * grad / (np.sqrt(self.G[i]) + self.eps)
```

## Adam（Adaptive Moment Estimation）

### 核心思想

结合 Momentum 和 RMSprop：

**一阶矩估计（动量）**：

$$m_t = \beta_1 m_{t-1} + (1-\beta_1) \nabla J$$

**二阶矩估计（自适应学习率）**：

$$v_t = \beta_2 v_{t-1} + (1-\beta_2)(\nabla J)^2$$

**偏差修正**：

$$\hat{\text{m}}_t = \frac{m_t}{1-\beta_1^t}, \quad \hat{v}_t = \frac{v_t}{1-\beta_2^t}$$

**参数更新**：

$$\theta_{t+1} = \theta_t - \frac{\alpha \hat{\text{m}}_t}{\sqrt{\hat{v}_t} + \epsilon}$$

```python
class Adam:
    def __init__(self, params, lr=0.001, beta1=0.9, beta2=0.999, eps=1e-8):
        self.params = params
        self.lr = lr
        self.beta1 = beta1
        self.beta2 = beta2
        self.eps = eps
        self.m = [np.zeros_like(p) for p in params]
        self.v = [np.zeros_like(p) for p in params]
        self.t = 0
    
    def step(self, grads):
        self.t += 1
        
        for i, (param, grad) in enumerate(zip(self.params, grads)):
            # 一阶矩
            self.m[i] = self.beta1 * self.m[i] + (1 - self.beta1) * grad
            
            # 二阶矩
            self.v[i] = self.beta2 * self.v[i] + (1 - self.beta2) * grad ** 2
            
            # 偏差修正
            m_hat = self.m[i] / (1 - self.beta1 ** self.t)
            v_hat = self.v[i] / (1 - self.beta2 ** self.t)
            
            # 更新
            param -= self.lr * m_hat / (np.sqrt(v_hat) + self.eps)
```

### 超参数

| 参数 | 默认值 | 含义 |
|------|--------|------|
| lr | 0.001 | 学习率 |
| beta1 | 0.9 | 一阶矩衰减率 |
| beta2 | 0.999 | 二阶矩衰减率 |
| eps | 1e-8 | 防止除零 |

## 优化器对比

| 优化器 | 动量 | 自适应学习率 | 偏差修正 | 推荐使用场景 |
|--------|------|-----------|---------|------------|
| SGD | 无 | 无 | 无 | 简单问题，大batch |
| Momentum | 有 | 无 | 无 | 加速收敛 |
| NAG | 有（Nesterov） | 无 | 无 | 需要更稳定收敛 |
| AdaGrad | 无 | 有（累积） | 无 | 稀疏数据 |
| RMSprop | 无 | 有（移动平均） | 无 | RNN，非平稳目标 |
| Adam | 有 | 有 | 有 | **默认选择** |

## 优化算法演变历史

### 从 SGD 到 Adam 的演进脉络

优化算法的发展经历了从简单到复杂、从通用到自适应的过程：

**第一代：基础梯度下降**
- **SGD（1951）**：最基础的优化方法，每次只用一个样本更新参数
- **Batch Gradient Descent**：使用全部数据计算梯度，稳定但慢
- **Mini-batch SGD**：折中方案，成为深度学习标准

**第二代：引入动量**
- **Momentum（1964）**：模拟物理动量，加速收敛、抑制震荡
- **Nesterov Accelerated Gradient（1983）**：先"看一步"再更新，更稳定

**第三代：自适应学习率**
- **AdaGrad（2011）**：根据历史梯度平方和自适应调整学习率
- **RMSprop（2012）**：使用指数移动平均替代累积，解决学习率单调递减问题
- **Adam（2014）**：结合动量和自适应学习率，成为最通用的优化器
- **AdamW（2017）**：改进权重衰减方式，与梯度更新解耦

**选择建议**：
- 大多数情况直接使用 **Adam** 或 **AdamW**
- 需要精细调参时尝试 **SGD + Momentum**
- RNN 类任务可尝试 **RMSprop**
- Transformer 训练优先使用 **AdamW**

## 学习率调度

### 固定学习率

最简单，但可能不是最优。

### 步长衰减

每隔一定步数，学习率乘以衰减系数：

```python
def step_decay(epoch, initial_lr=0.1, drop=0.5, epochs_drop=10):
    return initial_lr * (drop ** (epoch // epochs_drop))
```

### 指数衰减

$$\alpha_t = \alpha_0 \cdot e^{-kt}$$

```python
def exp_decay(epoch, initial_lr=0.1, k=0.1):
    return initial_lr * np.exp(-k * epoch)
```

### 余弦退火

$$\alpha_t = \alpha_{min} + \frac{1}{2}(\alpha_{max} - \alpha_{min})(1 + \cos(\frac{t}{T}\pi))$$

```python
def cosine_annealing(epoch, T_max, eta_min=0, eta_max=0.1):
    return eta_min + (eta_max - eta_min) * (1 + np.cos(np.pi * epoch / T_max)) / 2
```

### 学习率预热

训练初期从小学习率开始，逐渐增大到目标值：

```python
def warmup_lr(epoch, warmup_epochs, target_lr):
    if epoch < warmup_epochs:
        return target_lr * (epoch + 1) / warmup_epochs
    return target_lr
```

## PyTorch 中的优化器

PyTorch 的优化器都继承自 `Optimizer` 基类，主要包含三个属性：
- `defaults`：存储优化器超参数（如 lr、momentum 等）
- `state`：参数的缓存（如动量缓冲、二阶矩估计等）
- `param_groups`：管理的参数组，可以分组设置不同学习率

```python
import torch
import torch.optim as optim

# 定义模型
model = torch.nn.Linear(10, 1)

# 不同优化器
sgd = optim.SGD(model.parameters(), lr=0.01, momentum=0.9)
momentum = optim.SGD(model.parameters(), lr=0.01, momentum=0.9)
adam = optim.Adam(model.parameters(), lr=0.001, betas=(0.9, 0.999))
adamw = optim.AdamW(model.parameters(), lr=0.001, weight_decay=0.01)
rmsprop = optim.RMSprop(model.parameters(), lr=0.01, alpha=0.99)

# 学习率调度器
scheduler = optim.lr_scheduler.StepLR(adam, step_size=10, gamma=0.1)
scheduler = optim.lr_scheduler.ExponentialLR(adam, gamma=0.95)
scheduler = optim.lr_scheduler.CosineAnnealingLR(adam, T_max=100)

# 训练循环
for epoch in range(100):
    # 前向传播
    output = model(input)
    loss = criterion(output, target)
    
    # 反向传播
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    
    # 更新学习率
    scheduler.step()
```

### 优化器使用要点

1. **必须实例化**：优化器是一个类，需要实例化后才能使用
2. **梯度置零**：每次 backward 前需要 `optimizer.zero_grad()`，因为 PyTorch 梯度不自动清零
3. **参数分组**：可以给不同层设置不同学习率

```python
from torchvision.models import resnet18
net = resnet18()

# 给不同层分配不同学习率
optimizer = optim.SGD([
    {'params': net.fc.parameters()},           # 全连接层使用默认 lr
    {'params': net.layer4[0].conv1.parameters(), 'lr': 1e-2}  # layer4 使用更大 lr
], lr=1e-5)
```

## AdamW

Adam 的改进版，将权重衰减与梯度更新解耦：

```python
# PyTorch 中直接使用
optimizer = torch.optim.AdamW(model.parameters(), lr=0.001, weight_decay=0.01)
```

**与 Adam 的区别**：
- Adam：$$\theta = \theta - \alpha \cdot \frac{\text{m}}{\sqrt{v}} - \alpha \lambda \theta$$
- AdamW：$$\theta = \theta - \alpha \cdot \frac{\text{m}}{\sqrt{v}} - \alpha \lambda \theta$$（权重衰减在参数更新时应用）

实际效果：AdamW 通常比 Adam 效果更好，是 Transformer 训练的标准选择。

## 小结

| 优化器 | 核心特点 | 推荐场景 |
|--------|---------|---------|
| SGD + Momentum | 简单有效 | 大batch，需要精细调参 |
| Adam | 自适应，易用 | **默认首选** |
| AdamW | 改进的权重衰减 | Transformer，NLP |
| RMSprop | 适合非平稳目标 | RNN |

## 优化器可视化对比

不同优化器在相同任务上的收敛行为差异显著：

| 优化器 | 收敛速度 | 稳定性 | 最终精度 | 适用场景 |
|--------|---------|--------|---------|---------|
| SGD | 慢 | 中等 | 高（调参后） | 大batch，精细调参 |
| Momentum | 中等 | 好 | 高 | 通用加速 |
| Adam | 快 | 好 | 高 | **默认首选** |
| AdamW | 快 | 好 | 更高 | Transformer，NLP |
| RMSprop | 中等 | 好 | 中等 | RNN，序列任务 |

### 实验观察要点

在训练过程中建议监控以下指标：
1. **训练 Loss 曲线**：观察收敛速度和震荡情况
2. **验证 Loss 曲线**：判断是否过拟合
3. **学习率变化**：确认调度器按预期工作
4. **梯度范数**：检查梯度爆炸或消失

```python
# 使用 TensorBoard 记录训练过程
from torch.utils.tensorboard import SummaryWriter
writer = SummaryWriter('runs/experiment_1')

for epoch in range(num_epochs):
    for i, (inputs, labels) in enumerate(train_loader):
        # ... 训练代码 ...
        
        # 记录学习率
        writer.add_scalar('Learning Rate', optimizer.param_groups[0]['lr'], epoch)
        # 记录损失
        writer.add_scalar('Loss/train', loss.item(), epoch * len(train_loader) + i)
        # 记录梯度范数
        total_norm = 0
        for p in model.parameters():
            if p.grad is not None:
                total_norm += p.grad.data.norm(2).item() ** 2
        writer.add_scalar('Gradient Norm', total_norm ** 0.5, epoch * len(train_loader) + i)
```
