# 17 批归一化与Dropout

批归一化（Batch Normalization）和 Dropout 是深度学习中最常用的两种正则化/加速技术。

## 批归一化（Batch Normalization）

### 问题：内部协变量偏移

深度网络中，每一层的输入分布会随着前面层参数的变化而变化，导致训练困难。

### 为什么需要批归一化

| 问题 | 描述 | 影响 |
|------|------|------|
| **内部协变量偏移** | 每层输入分布变化 | 需要低学习率，训练慢 |
| **梯度消失** | 深层网络梯度变小 | 前面层难以训练 |
| **初始化敏感** | 对初始权重敏感 | 需要精心初始化 |
| **学习率限制** | 高学习率导致不稳定 | 收敛慢 |

### 核心思想

对每一层的输入进行归一化，使其均值为0，方差为1：

$$\hat{x} = \frac{x - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}$$

然后学习缩放和平移参数：

$$y = \gamma \hat{x} + \beta$$

### 批归一化的优势

1. **加速训练**：允许使用更大的学习率
2. **减少初始化依赖**：对初始权重不那么敏感
3. **轻微正则化**：每个batch的统计量不同，有噪声
4. **允许更高学习率**：训练更快收敛

```python
import numpy as np

class BatchNorm:
    def __init__(self, num_features, eps=1e-5, momentum=0.1):
        self.eps = eps
        self.momentum = momentum
        
        # 可学习参数
        self.gamma = np.ones((1, num_features))
        self.beta = np.zeros((1, num_features))
        
        # 运行时统计量
        self.running_mean = np.zeros((1, num_features))
        self.running_var = np.ones((1, num_features))
    
    def forward(self, x, training=True):
        if training:
            # 计算当前 batch 的均值和方差
            mean = np.mean(x, axis=0, keepdims=True)
            var = np.var(x, axis=0, keepdims=True)
            
            # 更新运行时统计量
            self.running_mean = (1 - self.momentum) * self.running_mean + self.momentum * mean
            self.running_var = (1 - self.momentum) * self.running_var + self.momentum * var
        else:
            # 使用运行时统计量
            mean = self.running_mean
            var = self.running_var
        
        # 归一化
        x_norm = (x - mean) / np.sqrt(var + self.eps)
        
        # 缩放和平移
        out = self.gamma * x_norm + self.beta
        
        return out

# 测试
bn = BatchNorm(4)
x = np.random.randn(32, 4)
out = bn.forward(x, training=True)
print(f"输入均值: {np.mean(x, axis=0)}")
print(f"输出均值: {np.mean(out, axis=0)}")
print(f"输出标准差: {np.std(out, axis=0)}")
```

### 为什么有效

1. **允许更大的学习率**：归一化后梯度更稳定
2. **减少对初始化的依赖**：不那么敏感
3. **轻微的正则化效果**：每个样本的归一化依赖于同 batch 的其他样本

### 深入理解：内部协变量偏移

深度网络中，每一层的输入分布会随着前面层参数的变化而变化，这种现象称为**内部协变量偏移（Internal Covariate Shift）**。BatchNorm 通过在每一层对输入进行归一化，使得每层的输入分布保持稳定，从而：

- **加速训练**：稳定的分布使得可以使用更大的学习率
- **允许更高学习率**：不用担心梯度爆炸或消失
- **减少对初始化的敏感性**：即使初始化不太理想也能较快收敛
- **轻微正则化**：每个样本的归一化依赖于同 batch 的其他样本，引入了一定的噪声

### 训练与测试的差异

BatchNorm 在训练和测试时的行为不同，这是理解它的关键：

**训练时**：
- 使用当前 batch 的均值和方差进行归一化
- 同时更新运行均值和运行方差（用于测试）

**测试时**：
- 使用训练阶段累积的运行均值和运行方差
- 不再计算当前 batch 的统计量

```python
# PyTorch 中自动处理训练和测试的差异
model.train()   # 训练模式：使用 batch 统计量
model.eval()    # 评估模式：使用 running_mean/var
```

### 使用位置

通常在**卷积层/全连接层之后，激活函数之前**：

```
Conv → BatchNorm → ReLU
```

### PyTorch 实现

```python
import torch
import torch.nn as nn

class NetWithBN(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv1 = nn.Conv2d(1, 32, 3)
        self.bn1 = nn.BatchNorm2d(32)
        self.conv2 = nn.Conv2d(32, 64, 3)
        self.bn2 = nn.BatchNorm2d(64)
        self.fc = nn.Linear(64 * 12 * 12, 10)
    
    def forward(self, x):
        x = self.conv1(x)
        x = self.bn1(x)
        x = torch.relu(x)
        
        x = self.conv2(x)
        x = self.bn2(x)
        x = torch.relu(x)
        
        x = x.view(x.size(0), -1)
        x = self.fc(x)
        return x
```

## Dropout

### 核心思想

训练时随机"丢弃"一部分神经元，防止它们共适应：

```
训练时：每个神经元以概率 p 被丢弃
测试时：使用所有神经元，输出乘以 (1-p)
```

### 直观理解

- **集成学习视角**：每次训练不同的子网络，最终是多个子网络的平均
- **鲁棒性视角**：神经元不能依赖其他特定神经元，必须学会独立工作

### 实现

```python
class Dropout:
    def __init__(self, p=0.5):
        self.p = p
    
    def forward(self, x, training=True):
        if training:
            # 生成掩码
            mask = (np.random.rand(*x.shape) > self.p).astype(float)
            # 缩放（保持期望不变）
            return x * mask / (1 - self.p)
        else:
            return x

# 测试
dropout = Dropout(p=0.5)
x = np.ones((1, 10))

print("训练时:")
for _ in range(3):
    print(dropout.forward(x, training=True))

print("\n测试时:")
print(dropout.forward(x, training=False))
```

### 常用 Dropout 率

| 层类型 | 推荐 Dropout 率 |
|--------|---------------|
| 输入层 | 0.2 |
| 隐藏层 | 0.3 - 0.5 |
| 卷积层 | 较少使用（BatchNorm 已足够） |
| RNN | 0.2 - 0.3（使用 Variational Dropout） |

### PyTorch 实现

```python
class NetWithDropout(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 256)
        self.dropout1 = nn.Dropout(0.5)
        self.fc2 = nn.Linear(256, 128)
        self.dropout2 = nn.Dropout(0.5)
        self.fc3 = nn.Linear(128, 10)
    
    def forward(self, x):
        x = x.view(x.size(0), -1)
        
        x = torch.relu(self.fc1(x))
        x = self.dropout1(x)  # 训练时自动丢弃，测试时自动关闭
        
        x = torch.relu(self.fc2(x))
        x = self.dropout2(x)
        
        x = self.fc3(x)
        return x

# 测试
model = NetWithDropout()
model.train()   # 训练模式（Dropout 启用）
model.eval()    # 评估模式（Dropout 关闭）
```

## BatchNorm vs Dropout

| 特性 | BatchNorm | Dropout |
|------|-----------|---------|
| 主要作用 | 加速训练，稳定梯度 | 防止过拟合 |
| 机制 | 归一化 + 学习缩放 | 随机丢弃神经元 |
| 训练/测试差异 | 使用不同统计量 | 训练丢弃，测试全部使用 |
| 现代网络 | 几乎必用 | CNN 中较少用（BN已足够） |
| 同时使用 | 可以，但顺序重要 | 通常 BN 在前 |

### 同时使用 BatchNorm 和 Dropout 的注意事项

在现代 CNN 架构中，BatchNorm 已经提供了足够的正则化效果，通常不需要再使用 Dropout。但在全连接层中，两者可以结合使用：

**推荐顺序**：
```
Conv → BatchNorm → ReLU → Dropout（可选）
```

**不推荐顺序**：
```
Dropout → BatchNorm   # 可能导致方差偏移
```

研究表明，在 BatchNorm 之后使用 Dropout 可能会导致性能下降，因为 Dropout 会改变 batch 的统计特性，影响 BatchNorm 的效果。因此，现代网络（如 ResNet）通常只使用 BatchNorm，而不使用 Dropout。

## 层归一化（Layer Normalization）

### 与 BatchNorm 的区别

| 特性 | BatchNorm | LayerNorm |
|------|-----------|-----------|
| 归一化维度 | 跨 batch（对每个特征） | 跨特征（对每个样本） |
| 依赖 batch size | 是 | 否 |
| 适合 | CNN | RNN, Transformer |
| 计算 | 需要维护运行时统计量 | 每个样本独立计算 |

```python
class LayerNorm:
    def __init__(self, num_features, eps=1e-5):
        self.eps = eps
        self.gamma = np.ones((1, num_features))
        self.beta = np.zeros((1, num_features))
    
    def forward(self, x):
        # 对每个样本归一化
        mean = np.mean(x, axis=1, keepdims=True)
        var = np.var(x, axis=1, keepdims=True)
        
        x_norm = (x - mean) / np.sqrt(var + self.eps)
        return self.gamma * x_norm + self.beta

# PyTorch
ln = nn.LayerNorm(normalized_shape=128)
```

## 完整示例

```python
import torch
import torch.nn as nn

class ModernNet(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()
        
        # 卷积层：Conv + BN + ReLU
        self.features = nn.Sequential(
            nn.Conv2d(1, 32, 3, padding=1),
            nn.BatchNorm2d(32),
            nn.ReLU(),
            nn.MaxPool2d(2),
            
            nn.Conv2d(32, 64, 3, padding=1),
            nn.BatchNorm2d(64),
            nn.ReLU(),
            nn.MaxPool2d(2),
            
            nn.Conv2d(64, 128, 3, padding=1),
            nn.BatchNorm2d(128),
            nn.ReLU(),
            nn.MaxPool2d(2),
        )
        
        # 全连接层：FC + Dropout
        self.classifier = nn.Sequential(
            nn.Linear(128 * 3 * 3, 256),
            nn.ReLU(),
            nn.Dropout(0.5),
            
            nn.Linear(256, 128),
            nn.ReLU(),
            nn.Dropout(0.3),
            
            nn.Linear(128, num_classes)
        )
    
    def forward(self, x):
        x = self.features(x)
        x = x.view(x.size(0), -1)
        x = self.classifier(x)
        return x

# 测试
model = ModernNet()
x = torch.randn(4, 1, 28, 28)

model.train()
out_train = model(x)
print(f"训练输出: {out_train.shape}")

model.eval()
out_eval = model(x)
print(f"评估输出: {out_eval.shape}")
```

## 小结

| 技术 | 作用 | 使用位置 | 注意点 |
|------|------|---------|--------|
| BatchNorm | 加速训练，稳定梯度 | Conv/FC 后，激活前 | 训练和测试行为不同 |
| LayerNorm | 同 BN，但不依赖 batch | RNN, Transformer | 每个样本独立 |
| Dropout | 防止过拟合 | FC 层之间 | 训练和测试行为不同 |
