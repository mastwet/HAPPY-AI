# 02 线性代数基础

线性代数是机器学习的数学基础。理解向量、矩阵和张量，是掌握后续算法的必备条件。

## 为什么需要线性代数

### 从几何到代数

线性代数提供了一套强大的工具，将几何直觉转化为代数运算：

| 几何概念 | 代数表示 | 应用场景 |
|---------|---------|---------|
| 点 | 向量 | 数据点、特征向量 |
| 线/平面 | 线性方程组 | 决策边界、回归线 |
| 旋转/缩放 | 矩阵乘法 | 图像变换、特征提取 |
| 投影 | 矩阵分解 | 降维、推荐系统 |

### 线性代数在机器学习中的应用

1. **数据表示**：每个数据样本都是高维空间中的一个向量
2. **特征变换**：通过矩阵乘法实现特征的线性变换
3. **降维**：PCA、SVD等算法都依赖矩阵分解
4. **优化**：梯度下降中的梯度就是向量

## 向量（Vector）

### 定义

向量是一组有序的数字，可以表示为一个行向量或列向量。

$$\mathbf{v} = [v_1, v_2, \ldots, v_n]$$

### 向量的本质

向量不仅仅是数字的集合，它代表了一个**有方向和大小的量**：

- **几何向量**：空间中的有向线段，有方向和长度
- **代数向量**：n维空间中的点，或从原点到该点的位移
- **数据向量**：特征值的集合，如一个人的[年龄, 身高, 体重]

### 基本运算

**加法**：对应元素相加
$$[1, 2, 3] + [4, 5, 6] = [5, 7, 9]$$

**数乘**：每个元素乘以标量
$$2 \times [1, 2, 3] = [2, 4, 6]$$

**点积（内积）**：
$$\mathbf{a} \cdot \mathbf{b} = \sum_{i=1}^{n} a_i b_i$$

```python
import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

# 向量加法
print(a + b)        # [5 7 9]

# 数乘
print(2 * a)        # [2 4 6]

# 点积
print(np.dot(a, b))  # 32
```

## 矩阵（Matrix）

### 定义

矩阵是一个二维数组，有行和列。

$$\mathbf{A} = \begin{bmatrix} a_{11} & a_{12} \\ a_{21} & a_{22} \end{bmatrix}$$

### 基本运算

**矩阵乘法**：

$$\mathbf{C} = \mathbf{A} \times \mathbf{B}$$

其中 $$C_{ij} = \sum_{k} A_{ik} B_{kj}$$

```python
import numpy as np

A = np.array([[1, 2],
              [3, 4]])
B = np.array([[5, 6],
              [7, 8]])

# 矩阵乘法
C = np.dot(A, B)
print(C)
# [[19 22]
#  [43 50]]
```

**重要性质**：矩阵乘法不满足交换律
$$\mathbf{A} \times \mathbf{B} \neq \mathbf{B} \times \mathbf{A}$$

### 特殊矩阵

| 矩阵类型 | 定义 | 特点 |
|---------|------|------|
| 单位矩阵 $$\mathbf{I}$$ | 对角线为1，其余为0 | $$\mathbf{A} \times \mathbf{I} = \mathbf{A}$$ |
| 对角矩阵 | 非对角线元素为0 | 计算简单 |
| 转置矩阵 $$\mathbf{A}^T$$ | 行列互换 | $$(\mathbf{AB})^T = \mathbf{B}^T \mathbf{A}^T$$ |
| 逆矩阵 $$\mathbf{A}^{-1}$$ | 满足 $$\mathbf{A}\mathbf{A}^{-1} = \mathbf{I}$$ | 不是所有矩阵都可逆 |

## 张量（Tensor）

张量是向量和矩阵的高维推广。

| 维度 | 名称 | 例子 |
|------|------|------|
| 0维 | 标量（Scalar） | 5 |
| 1维 | 向量（Vector） | [1, 2, 3] |
| 2维 | 矩阵（Matrix） | [[1,2],[3,4]] |
| 3维+ | 张量（Tensor） | 图像数据 (H×W×C) |

```python
import numpy as np

# 标量
scalar = 5

# 向量
vector = np.array([1, 2, 3])

# 矩阵
matrix = np.array([[1, 2], [3, 4], [5, 6]])

# 张量（3维：2张3×3的图像）
tensor = np.random.rand(2, 3, 3)
print(tensor.shape)  # (2, 3, 3)
```

### 向量的几何意义

向量不仅有代数表示，还有直观的几何意义：
- **方向**：向量指向空间中的某个方向
- **长度（模）**：$$\vert\mathbf{v}\vert = \sqrt{\sum_{i=1}^{n} v_i^2}$$
- **夹角**：两个向量的点积与它们夹角的关系：$$\mathbf{a} \cdot \mathbf{b} = \vert\mathbf{a}\vert \vert\mathbf{b}\vert \cos	heta$$

**实际例子**：
- 在推荐系统中，用户兴趣可以表示为向量，两个用户的向量夹角越小，兴趣越相似
- 在词嵌入中，每个词是一个向量，语义相近的词向量距离更近

### 常见误区

1. **混淆点积和逐元素乘法**：点积结果是标量，逐元素乘法结果仍是向量
2. **忽视矩阵乘法的维度匹配**：$$A_{(m,n)} 	imes B_{(n,p)} = C_{(m,p)}$$，内维必须相等
3. **认为所有矩阵都有逆矩阵**：只有方阵且行列式不为零的矩阵才可逆
4. **不理解张量的实际意义**：3维张量可以表示彩色图像（高×宽×通道），4维张量可以表示一批图像（样本数×高×宽×通道）

### 与其他知识点的联系

- **线性回归**：正规方程 $$\mathbf{w} = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \mathbf{y}$$ 大量用到矩阵运算（见第06章）
- **PCA降维**：通过特征值分解实现（见第05章）
- **神经网络**：每一层都是矩阵乘法加激活函数
- **协方差矩阵**：描述特征之间的线性关系（见第03章）

## 在机器学习中的应用

### 数据表示

- **特征向量**：每个样本表示为一个向量
- **数据集**：多个样本堆叠成矩阵
- **图像数据**：3维张量 (高度 × 宽度 × 通道)

```python
# 数据集表示
# 100个样本，每个样本有10个特征
X = np.random.rand(100, 10)  # 形状: (100, 10)

# 100个标签
y = np.random.randint(0, 2, 100)  # 形状: (100,)

# 图像数据：32张32×32的RGB图像
images = np.random.rand(32, 32, 32, 3)  # 形状: (32, 32, 32, 3)
```

### 线性变换

神经网络中的每一层本质上都是线性变换 + 非线性激活：

$$\mathbf{y} = \mathbf{W}\mathbf{x} + \mathbf{b}$$

其中：
- $$\mathbf{W}$$ 是权重矩阵
- $$\mathbf{x}$$ 是输入向量
- $$\mathbf{b}$$ 是偏置向量
- $$\mathbf{y}$$ 是输出向量

## 小结

| 概念 | 含义 | 在ML中的作用 |
|------|------|-------------|
| 向量 | 一维数组 | 特征表示、权重 |
| 矩阵 | 二维数组 | 数据集、权重矩阵 |
| 张量 | 高维数组 | 图像、批量数据 |
| 矩阵乘法 | 线性变换 | 神经网络前向传播 |
