# 03 概率与统计基础

概率论与统计学是机器学习的理论基础。理解随机性、不确定性和数据分布，是构建可靠模型的前提。

## 概率的基本概念

### 概率的本质

概率是描述**不确定性**的数学工具。在机器学习中，几乎所有问题都涉及不确定性：

- 数据本身有噪声和随机性
- 模型参数需要通过数据推断
- 预测结果总是带有一定的不确定性

### 频率学派 vs 贝叶斯学派

| 学派 | 核心观点 | 概率解释 | 代表方法 |
|------|---------|---------|---------|
| **频率学派** | 概率是长期频率 | 事件在重复试验中的发生比例 | 最大似然估计 |
| **贝叶斯学派** | 概率是信念程度 | 对事件发生的主观置信度 | 贝叶斯推断 |

**例子**：抛硬币
- 频率学派：抛10000次，正面朝上5003次，概率≈0.5
- 贝叶斯学派：根据先验知识（硬币看起来均匀），认为正面概率约为0.5

### 随机事件与概率

**概率**描述事件发生的可能性，取值范围 $$[0, 1]$$。

$$P(A) = \frac{\text{事件}A\text{发生的有利情况数}}{\text{所有可能情况数}}$$

**基本性质**：
- $$0 \leq P(A) \leq 1$$
- $$P(\Omega) = 1$$（必然事件概率为1）
- $$P(\emptyset) = 0$$（不可能事件概率为0）

### 条件概率

在事件 **B** 发生的条件下，事件 **A** 发生的概率：

$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

**贝叶斯定理**：

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

贝叶斯定理是朴素贝叶斯分类器的理论基础。

## 随机变量

### 离散型随机变量

取值有限或可数无限的随机变量。

**常见分布**：

| 分布 | 描述 | 例子 |
|------|------|------|
| 伯努利分布 | 单次试验，成功/失败 | 抛硬币 |
| 二项分布 | n次独立伯努利试验的成功次数 | n次抛硬币中正面的次数 |
| 泊松分布 | 单位时间/空间内随机事件发生次数 | 每小时接到的电话数 |

### 连续型随机变量

取值连续的随机变量。

**概率密度函数（PDF）**：

$$P(a \leq X \leq b) = \int_a^b f(x) dx$$

**常见分布**：

| 分布 | 描述 | 例子 |
|------|------|------|
| 均匀分布 | 区间内等概率 | 随机数生成 |
| 正态分布（高斯分布） | 钟形曲线 | 身高、测量误差 |
| 指数分布 | 等待时间 | 设备故障间隔时间 |

### 正态分布（重点）

正态分布是机器学习中最重要的分布：

$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

其中 $$\mu$$ 是均值，$$\sigma$$ 是标准差。

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成正态分布数据
mu, sigma = 0, 1
samples = np.random.normal(mu, sigma, 1000)

plt.hist(samples, bins=30, density=True, alpha=0.7)
plt.title("Normal Distribution: μ=0, σ=1")
plt.xlabel("x")
plt.ylabel("Probability Density")
plt.show()
```

**68-95-99.7 法则**：
- 约 68% 的数据落在 $$[\mu-\sigma, \mu+\sigma]$$ 内
- 约 95% 的数据落在 $$[\mu-2\sigma, \mu+2\sigma]$$ 内
- 约 99.7% 的数据落在 $$[\mu-3\sigma, \mu+3\sigma]$$ 内

## 期望与方差

### 期望（Expected Value）

随机变量的平均值：

$$E[X] = \sum_i x_i P(X=x_i) \quad \text{（离散）}$$

$$E[X] = \int_{-\infty}^{\infty} x f(x) dx \quad \text{（连续）}$$

**性质**：
- $$E[aX + b] = aE[X] + b$$
- $$E[X + Y] = E[X] + E[Y]$$

### 方差（Variance）

衡量随机变量偏离期望的程度：

$$Var(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2$$

**标准差**：$$\sigma = \sqrt{Var(X)}$$

```python
import numpy as np

data = np.array([1, 2, 3, 4, 5])

# 期望（均值）
mean = np.mean(data)
print(f"均值: {mean}")  # 3.0

# 方差
variance = np.var(data)
print(f"方差: {variance}")  # 2.0

# 标准差
std = np.std(data)
print(f"标准差: {std}")  # 1.414...
```

## 最大似然估计（MLE）

### 核心思想

给定观测数据，找到最可能产生这些数据的参数。

### 例子：估计正态分布参数

假设观测到数据 $$x_1, x_2, \ldots, x_n$$，假设它们来自正态分布 $$N(\mu, \sigma^2)$$。

**似然函数**：

$$L(\mu, \sigma) = \prod_{i=1}^n \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x_i-\mu)^2}{2\sigma^2}}$$

**对数似然**：

$$\log L(\mu, \sigma) = \sum_{i=1}^n \left[ -\log(\sqrt{2\pi}\sigma) - \frac{(x_i-\mu)^2}{2\sigma^2} \right]$$

**求导并令其为零**，得到：

$$\hat{\mu} = \frac{1}{n}\sum_{i=1}^n x_i \quad \text{（样本均值）}$$

$$\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \hat{\mu})^2 \quad \text{（样本方差）}$$

```python
import numpy as np

# 观测数据
np.random.seed(42)
true_mu, true_sigma = 5, 2
data = np.random.normal(true_mu, true_sigma, 1000)

# MLE 估计
mu_mle = np.mean(data)
sigma_mle = np.std(data)

print(f"真实值: μ={true_mu}, σ={true_sigma}")
print(f"MLE估计: μ={mu_mle:.3f}, σ={sigma_mle:.3f}")
```

### 通俗理解概率

概率描述的是**不确定性**。生活中充满了不确定性：
- 明天下雨的概率是60%
- 掷骰子出现6点的概率是1/6
- 某封邮件是垃圾邮件的概率是85%

机器学习模型输出的概率，本质上是对不确定性的量化估计。

### 常见分布的直观理解

| 分布 | 直观理解 | 实际例子 |
|------|---------|---------|
| 伯努利分布 | 抛一次硬币 | 邮件是否为垃圾邮件 |
| 二项分布 | 抛n次硬币，正面出现k次 | n次点击中成功转化的次数 |
| 泊松分布 | 单位时间内随机事件发生次数 | 每小时网站访问次数 |
| 正态分布 | 自然界最常见的分布 | 身高、考试成绩、测量误差 |
| 指数分布 | 等待时间 | 设备故障间隔时间 |

### 常见误区

1. **混淆概率和频率**：频率是过去发生次数的比例，概率是对未来的预测。抛100次硬币出现48次正面，频率是0.48，但下一次抛硬币正面的概率仍是0.5
2. **忽视条件概率的方向性**：3549P(A|B) 
eq P(B|A)3549，这是贝叶斯定理的核心
3. **误解大数定律**：大数定律说的是大量重复试验后频率趋近概率，不是下次该出现了
4. **混淆期望和均值**：样本均值是估计量，期望是理论值。当样本量趋近无穷时，样本均值收敛于期望

### 与其他知识点的联系

- **逻辑回归**：输出的是概率值，用极大似然估计求解（见第07章）
- **朴素贝叶斯**：基于贝叶斯定理的分类器
- **高斯混合模型**：用正态分布的混合来聚类
- **权重初始化**：Xavier/He初始化假设权重服从特定分布

## 在机器学习中的应用

| 概念 | 应用 |
|------|------|
| 概率 | 分类器的输出（softmax概率） |
| 期望 | 损失函数的期望风险最小化 |
| 方差 | 特征缩放、数据标准化 |
| 正态分布 | 权重初始化（如Xavier、He初始化） |
| 最大似然 | 逻辑回归、高斯混合模型 |
| 贝叶斯定理 | 朴素贝叶斯分类器 |

## 小结

| 概念 | 含义 | ML应用 |
|------|------|--------|
| 概率 | 事件发生的可能性 | 模型输出置信度 |
| 期望 | 随机变量的平均值 | 损失函数 |
| 方差 | 数据的离散程度 | 特征标准化 |
| 正态分布 | 最常见的数据分布 | 初始化、假设检验 |
| MLE | 从数据估计参数 | 模型参数估计 |
