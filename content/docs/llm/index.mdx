# 第三篇：大语言模型

从 NLP 基础到 Transformer，从预训练到 RAG/Agent——完整覆盖 LLM 技术栈。

本篇是整个教程的**最终目标篇**。基础篇（1-10 章）帮你打好数学和经典 ML 基础，进阶篇（11-24 章）带你掌握深度学习和生成模型，而本篇将带你**从零理解并动手实现大语言模型**。

